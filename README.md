# nodejs-basic-server
This repository contains my Node.js learning journey, including basic server setup, routing with multiple endpoints, file system operations, and fundamental JavaScript concepts.
# Node.js Basics 🚀

This repository contains my beginner-level practice using Node.js.

## 📌 Topics Covered

* JavaScript basics (variables, functions, conditions)
* Node.js modules using require()
* File system operations (fs module)
* Creating a basic server
* Routing with multiple URLs
* Asynchronous behavior (setTimeout)

## 🌐 Server Routes

* / → Home page
* /about → About page
* /login → Login page
* Other → 404 Not Found

## ▶️ How to Run

Run any file using:

bash
node filename.js


Example:

bash
node server-routing.js


Then open:
http://localhost:3000
